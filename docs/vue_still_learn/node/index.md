### console

| console.dir                                          | 将对象的信息输出到控制台 |
| ---------------------------------------------------- | ------------------------ |
| console.time和console.timeEnd                        | 统计代码运行时间         |
| console.log,console.error,console.error,console.info | 打印输出                 |
| console.trace                                        | 打印栈信息               |
| console.table                                        | 数组格式信息表格形式输出 |
