### 3.1一个简单的Java应用程序
- FirstSample.java
```java
public class FirstSample{
    public static void main(Strong[] args){
        System.out.printin("We will not use 'Hello World!'")
    }
}
```
::: tip 注
- public 访问修饰符 控制程序的其他部分对这段代码的访问级别
- class 类 加载程序逻辑的容器
- java中的main方法必须是静态的
- void表示该方法无返回值
- System.out还有一个print方法，输出之后不换行
:::
